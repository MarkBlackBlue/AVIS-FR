# AVIS Frosinone - Restyling 2025
## Progetto
Sito istituzionale per AVIS Comunale di FROSINONE.

## Stack Tecnico
- **PHP**: 8.2+
- **Joomla**: 5.3.x LTS
- **Helix Ultimate**: 2.1.x
- **MySQL**: 8.0

## Setup Locale (XAMPP/Docker)
1. Clona il repository.
2. Scarica Joomla 5.3.x e installalo nella root.
3. Installa Helix Ultimate.
4. Sovrascrivi le cartelle `templates/helixultimate/` con i file presenti in `src/template/`.
5. Carica `custom.css` e `slider.js` nelle rispettive cartelle.
6. Importa lo schema SQL iniziale da `database/`.
